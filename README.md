# Código del curso — Spec-Driven Development con Python, GitHub Copilot, FastAPI y PostgreSQL

Este repositorio contiene el código de los vídeos del curso que incluyen código descargable. La nomenclatura de cada zip es:

```
S[tema]E[vídeo].zip
```

Por ejemplo, `S03E01.zip` corresponde al **tema 3, vídeo 1**.

No todos los vídeos tienen código asociado: algunos son teóricos o de configuración del entorno y no generan un proyecto descargable.
